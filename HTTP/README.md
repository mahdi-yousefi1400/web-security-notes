# HTTP Basics

## What is HTTP?
HyperText Transfer Protocol used for communication between client and server.

## Request Structure
- Method
- Path
- Headers
- Body

## Response Structure
- Status Code
- Headers
- Body

## Important Status Codes
200 OK
301 Redirect
302 Found
403 Forbidden
404 Not Found
500 Internal Server Error

## Practical Practice
- Used browser DevTools to inspect requests
- Observed headers and status codes
